Where is interview Page, where can I make interview?

Go to main page. Click "Application to Review" then you reach with a adverts list page. Select a advert and click "See Applications" You will see a application List now, and select a application and make interview.

Home Page + Application to Review(button) -> Application to Review Page + See Applications(Button) -> Interview Page

Where can I find adverts?

You can find all adverts in bottom button which name is "See Your Department Adverts" in main page

Home Page + "See Your Department Adverts"(button) -> All Adverts Page
