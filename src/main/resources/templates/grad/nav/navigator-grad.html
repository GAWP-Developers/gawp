Where is application control page?

Firstly, please sure be in home page. If not when you can press top to logo that is "Graduate Application Program", you will go to home page. After that please find "Applications Before Forwarding to Department" button and press it. Then you find "Application to Preview" and "Verify Approved Applications". Enjoy it!

Home Page + Applications Before Forwarding to Department(button) -> Applications Before Forwarding to Department Page +Applications to Pre-Review -> Applications to Pre-Review Page

Where can I find application after department reviewd them?

Go to home page. Then, click "Applications From Department" button. You can see some adverts in this page. Select a advert, and then you can see the applications which are interviewd by the resposible dept.

Home Page + Applications From Department(button) ->Applications From Department Page -> Adverts + See Applications(button) -> Applications
