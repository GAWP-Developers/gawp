Where is advert apply page?

Firstly, please sure be in home page. If not when you can press top to logo that is "Graduate Application Program", you will go to home page. After that please find "Apply" button and press it. Then you will go to advert page. Please select a advert which is good for you think and press on it or it's See Details button. Here you can find details of the advert. If you press Apply then you reach "Application form". Enjoy it!

Home Page + Apply(button) -> Advert Page + See Details(button) -> Advert Details + Apply(button) -> Application Form

Where is my applications?

Firstly, please sure be in home page. If not, when you can press top to logo that is "Graduate Application Program",you will go to home page. After that please find "My Applications" button and press it.Enjoy it.

Home Page + My Applications(button) -> My Application Page

Where can I edit my profile informations?

In main page, you can find "Edit Profile" button from top right. If you are not in main page,you can press top to logo that is "Graduate Application Program",you will go to home page.

Home Page + Edit Profile(button) -> Edit Profile Page
